# ATM Report Generator

A comprehensive ATM error tracking and reporting system for GitHub Pages!

## 🚀 Quick Start

1. **Clone or fork** this repository
2. **Go to Settings → Pages**
3. **Enable GitHub Pages** from main branch
4. **Visit your site** after 2-5 minutes

## 🌐 Access Your System

Once deployed, visit:
```
https://yourusername.github.io/atm-report-generator/
```

## ✨ Features

- **6 Error Categories** with color coding
- **10 Reason Options** including Power problem
- **Mobile Responsive Design** for all devices
- **Advanced Search 
- **Analytics Dashboard** with charts
- **Data Export** in CSV format
- **Import/Export** for backup and restore
- **Touch-Friendly Interface** for mobile use

## 📱 Mobile Features

- Responsive design works on phones, tablets, desktops
- Touch-friendly buttons and navigation
- Hamburger menu for mobile navigation
- Optimized typography for small screens

## 💾 Data Storage

- **Local Storage** - Data saved in browser
- **Export Backup** - Save data as JSON
- **Import Restore** - Load from backup
- **Manual Sync** - Share files with team
