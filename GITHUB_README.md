# ATM Report Generator - GitHub Pages Version

A comprehensive ATM error tracking and reporting system that works on GitHub Pages!

## 🎯 Features

### ✅ Core Features
- **6 Error Categories** (Lost Communication, Hard Fault, Software Problem, Network Issue, Power Failure, Out Of Service)
- **10 Reason Options** (Moti case, Network problem, Branch Follow up problem, 2nd/3rd/4th remainder, Aptra Problem, Corridor, Cash Shortage, Power problem)
- **Mobile Responsive Design** (works on phones, tablets, desktops)
- **Advanced Search & Filtering** (real-time search across ATMs, branches, reasons)
- **Analytics Dashboard** (basic charts and metrics)
- **Data Export** (CSV format)
- **Import/Export** (JSON backup/restore)

### ✅ User Experience
- **Touch-Friendly Interface** (44px minimum touch targets)
- **Mobile Navigation** (hamburger menu for small screens)
- **Responsive Grid Layouts** (adapts to screen size)
- **Real-time Updates** (instant feedback)
- **Data Persistence** (browser local storage)

## 🚀 Quick Start

### Option 1: Use This Repository (Fastest)
1. **Click "Use this template"** or fork this repository
2. **Go to Settings → Pages**
3. **Enable GitHub Pages** from main branch
4. **Visit your site** after 2-5 minutes

### Option 2: Manual Setup
1. **Create new repository** named `atm-report-generator`

## 📱 How to Use

### Basic Operations:
1. **Add ATMs to Categories** - Click "Add ATMs" button in any category
2. **Select Reasons** - Choose from 10 available reason options
3. **Set Start DateTime** - Track when issues began
4. **Export Data** - Download CSV reports
5. **Search & Filter** - Find specific ATMs or issues

### Data Management:
- **Automatic Save** - All changes saved to browser storage
- **Export Backup** - Save data as JSON file
- **Import Restore** - Load data from backup file
- **Clear Filters** - Reset search and filters
